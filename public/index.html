Badger Press

Fill in the fields on the left to generate a badge image. The preview above will be updated in real-time as you type. Connect your badger over USB, then click the "Copy to Badge" button to copy the badge to your clipboard. Note that this requires the Web Serial API's which work in Chrome and Edge on the desktop, but not currently Safari

To learn more about your GitHub Badger 2350, visit the repo. You can also edit the code running on your Badger directly using the Viper Web IDE or you can install Thonny locally.

The GitHub Badger 2350 was created in partnership with the wonderful team at Pimoroni.
